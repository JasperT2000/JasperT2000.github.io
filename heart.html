Heart Attack Prediction using Machine Learning

Heart attack or Cardio-Vascular diseases are major cause for deaths in human being. In recent days, youngsters are facing heart diseases without any prior warning.This project aims to provide an alert to patients with the help of their lab data. The project comprises of

  • Dataset collection from kaggle
  • Exploratory data Analysis
  • Model development
  • Model deployment
  • Website creation

The main difference between this project and the other works are that, this system makes use of the probability of the predicted result tp provide in-depth result to the patient along with the risk factor. The code for this project can be found here..

ST Slope

The ST segment is the flat, isoelectric section of the ECG between the end of the S wave (the J point) and the beginning of the T wave.

From the above bar plot, we can find that there is clear distinguished pattern. The flat ST Slope signifies more risk of being a heart patient than the others.

Correlation heatmap

The heatmap is useful in terms of viewing the correlation between various variables (especially the target variable) which helps us draw conclusions on a simple level as how a variable is affected with the changes in other variable.

Model Development

For this task, I have chosen to proceed with Random Forest classifier. The dataset is scaled, pre-processed and then the model is being trained and tested. The results are shown below:

The model has performed well. The accuracy achieved by the model is around 85% which suggests that the model has fitted the data properly. The model is ready for deployment.

Model Deployment

To deploy the model, the model was converted to pickle format so that it can be used at the backend for a small website. The website has a form to get the input variables from the user. This data collected is then used to feed into the deployed model and the prediction is obtained. Based on the obtained prediction and its probability of occurrence, 4 different result pages have been developed go warn the user.

Home page:

The result pages that should be shown from one of the 4 shown below is chosen by the probability value returned from the model

  • 0 < Probability < 30 : Green zone
  • 30 < Probability < 50 : Yellow zone
  • 50 < Probability < 70 : Orange zone
  • 70 < Probability < 100 : Red zone
